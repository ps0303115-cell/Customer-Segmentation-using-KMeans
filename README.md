# Customer Segmentation using K-Means Clustering

## Overview

This project performs customer segmentation using the K-Means Clustering algorithm based on customers' annual income and spending score.

Developed as **Task-02** for the **Prodigy InfoTech Machine Learning Internship**.

## Dataset

Mall Customer Segmentation Dataset

https://www.kaggle.com/datasets/vjchoudhary7/customer-segmentation-tutorial-in-python

## Technologies Used

- Python
- Pandas
- Matplotlib
- Scikit-Learn

## Algorithm

K-Means Clustering

## Features Used

- Annual Income (k$)
- Spending Score (1-100)

## Output

- Elbow Method Graph
- Customer Clusters
- Cluster Centroids

## How to Run

pip install -r requirements.txt
python customer_segmentation.py
